## 🔍 *SCOPA (Searching for COde PArtener)*

**프로젝트**를 함께 할 **파트너**를 찾는 개발자 구인 **플랫폼** 서비스

간단한 **코드가 포함된 프로필**로 나와 스타일이 꼭 맞는 파트너를 찾아봐요!

# 🙌 배포주소
https://www.scopa.ga/

# 🎥 시연영상

# 🎞️ 주요기능
### 1. 소셜로그인
### 2. 회원가입
### 3. 프로필 수정
### 4. 필터링 및 페이징
### 5. 좋아요
### 6. 쪽지기능

# 💻 실행방법
### FrontEnd
```
cd client
npm i
npm run start
```
### BackEnd
```
cd server
npm i
npm run start
```
## (환경변수)
### client
```
REACT_APP_GOOGLE_OAUTH=
REACT_APP_GITHUB_OAUTH=
REACT_APP_FETCH_URL=
```
### server
```
GOOGLE_CLIENT_ID=
GOOGLE_CLIENT_SECRET=
GITHUB_CLIENT_ID=
GITHUB_CLIENT_SECRET=
MONGO_URL= 
DB_NAME=
CLIENT_URL=
SERVER_URL=
```
