# 💻 실행방법

### FrontEnd

```
cd client
npm i
npm run start
```

### BackEnd

```
cd server
npm i
npm run start
```

## 환경변수

### client

```
REACT_APP_GOOGLE_OAUTH=
REACT_APP_GITHUB_OAUTH=
REACT_APP_FETCH_URL=
```

### server

```
GOOGLE_CLIENT_ID=
GOOGLE_CLIENT_SECRET=
GITHUB_CLIENT_ID=
GITHUB_CLIENT_SECRET=
MONGO_URL=
DB_NAME=
CLIENT_URL=
SERVER_URL=
```
